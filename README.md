# Currency Converter Web Application

A responsive Currency Converter web application built using HTML, CSS, and JavaScript. The project fetches real-time exchange rates from a public API and allows users to convert between multiple international currencies with an intuitive and clean user interface.

---

## Overview

This project demonstrates practical frontend development skills, including API integration, DOM manipulation, asynchronous JavaScript, and responsive design. It focuses on building a real-world utility while following clean code practices.

---

## Features
- Real-time currency conversion using live exchange rates
- Supports multiple global currencies
- Dynamic country flag updates based on selected currency
- Input validation for invalid or empty values
- Responsive UI for mobile, tablet, and desktop screens
- Clear and user-friendly output display

---

## Tech Stack
- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (ES6)
- Public Currency Exchange API

---

## What I Learned
- Integrating third-party APIs using fetch and async/await
- Handling asynchronous data and API responses
- DOM traversal and dynamic UI updates
- Event handling for user interactions
- Input validation and basic error handling
- Building responsive layouts using media queries
